backend/app/api
本目录负责：HTTP 接口层，对外暴露 API。
不负责：核心业务规则、数据库模型定义。声明：一旦本目录结构或职责发生变化，请更新本文件。

文件说明
`assets.py` 地位：素材接口。职责：提供素材查询与管理相关 API。
`job_logs.py` 地位：任务日志接口。职责：按任务暴露执行日志查询能力。
`jobs.py` 地位：任务控制接口。职责：创建生成任务、查询状态、取消任务、更新任务状态。
`logs.py` 地位：AI 日志接口。职责：查询 AI 调用日志和统计信息。
`projects.py` 地位：项目接口。职责：创建项目、查询项目、查询项目结果。
`scenes.py` 地位：分镜接口。职责：查询与编辑分镜数据。
