backend/app/models
本目录负责：数据库持久化模型定义。
不负责：业务规则、HTTP 输入输出协议。声明：一旦本目录结构或职责发生变化，请更新本文件。

文件说明
`__init__.py` 地位：模型出口。职责：统一暴露模型给其他模块引用。
`ai_log.py` 地位：AI 日志模型。职责：持久化关键 AI 输入输出与状态。
`asset.py` 地位：素材模型。职责：记录项目素材及其元数据。
`generation_job.py` 地位：任务模型。职责：记录生成任务状态、阶段和结果。
`job_log.py` 地位：任务日志模型。职责：记录任务各阶段运行日志。
`project.py` 地位：项目模型。职责：存储文章项目基础信息。
`scene.py` 地位：分镜模型。职责：存储当前生效分镜内容。
`scene_version.py` 地位：分镜版本模型。职责：保存分镜编辑历史版本。
